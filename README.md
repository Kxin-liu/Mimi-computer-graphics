# Mimi-computer-graphics
计算机图形学大作业呜呜（《小咪duangduang乐》真的很Q弹

## 项目结构

### 1. 核心代码文件
- **main.cpp**：主程序，包含游戏循环、场景管理、输入处理
- **TriMesh.cpp**：几何体管理，生成和加载3D模型
- **MeshPainter.cpp**：绘制系统，管理着色器和渲染调用
- **Camera.h**：相机系统，处理视角和移动
- **BALLS.h**：台球物理系统，包含位置、速度、碰撞检测

### 2. 资源文件
- **shaders/**：GLSL着色器代码
- **assets/**：纹理、模型、图像资源

### 3. 构建系统
- **CMakeLists.txt**：CMake配置（环境配置微微微复杂，不过我这个VScode也可以运行哦~
